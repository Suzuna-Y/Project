# 学習資料に基づく生成AIを用いた問題生成・解説システムの試作
## システム概要
- 本システムは、PDFやPowerPoint形式の教材から生成AIを用いて四択問題を自動生成し、学習者の復習を支援するWebシステムです。
- 教育現場では、教員による問題作成に多くの時間と労力が必要であり、学習者も復習用の問題が不足する場合があります。そこで本システムでは、教材資料をアップロードするだけで問題を自動生成し、効率的な学習支援を実現することを目的としています。
- 生成された問題はWebブラウザ上で解答でき、正誤判定と解説を表示します。また、誤答した問題を記録し、苦手分野の把握や重点的な復習を支援します。
